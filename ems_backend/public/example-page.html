Example Page

This page demonstrates how easy it is to create new pages with the component system.

The sidebar and topbar are loaded dynamically and all functionality is preserved:

  • ✅ Role-based sidebar display
  • ✅ Sidebar toggle functionality
  • ✅ Active navigation highlighting
  • ✅ User role display in topbar
  • ✅ Navigation between pages

How to use:

  1. Include the components.js script
  2. Call loadPageComponents(title, currentPage) in DOMContentLoaded
  3. That's it! All functionality is automatically set up
